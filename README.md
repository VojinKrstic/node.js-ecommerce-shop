# Node.js Mini Shop

Mini e-commerce aplikacija razvijena tokom kursa
NodeJS - The Complete Guide (Maximilian Schwarzmüller).

## Features

- Authentication & sessions
- Product management (CRUD)
- Shopping cart
- Orders system
- File uploads (Multer)
- PDF invoice generation
- Stripe Checkout payment
- Pagination
- CSRF protection
- Form validation

## Tech Stack

- Node.js
- Express
- MongoDB + Mongoose
- EJS templating
- Stripe API
- Multer (file uploads)
- express-session + connect-mongo
- csurf
- bcrypt
- PDFKit
